一. Visual Studio 使用 Zeal 作为文档查看工具步骤：

1. 使用 “外部工具” 添加 Zeal 工具。


2. 为 Zeal 外部命令添加快捷键。


3. 在 Visual Studio 编辑器中选中需要查看文档的内容, 按下快捷键 “Shift+Atl+Z” 即会调用 Zeal 并定位到相应页面。

二. Vs Code 使用 Zeal 作为文档查看工具步骤：

1. 在扩展中搜索 “Dash” 并安装。


2. 修改文档工具插件的快捷键。


3. 在 Vs Code 编辑器中光标置于(或选中)需要查看文档的内容, 按下快捷键 “Shift+Atl+Z” 即会调用 Zeal 并定位到相应页面。
